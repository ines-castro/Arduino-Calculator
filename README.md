# Arduino Calculator
This repository contains the materials for a learning hub at [TUMO Portugal](https://tumo.pt) where kids and teens (12–18) build their own calculator using Arduino, covering electronics and programming. The project includes a comprehensive [Step-by-step guide](Guia_PT.pdf), only available in portuguese.

## Structure
This repository is divided into three main folders:

- **3D:** Contains files for 3D printing the calculator case. These parts are used to enclose the electronic components and turn the project into a physical calculator.

- **Electronics:** Includes the test file to help verify that the electronic components are correctly assembled and wired.

- **Programming:** Contains the Arduino code that makes the calculator work. The code is divided into two different difficulty levels: level 1 (core) and level 2 (advanced).
